# MERN Stack Boiler Plate

a simple pre-configured mern project

## RUN SERVER

change directory into server folder

```
	cd server && npm install
```

and run 

```
 	npm run dev
```

## START CLIENT

open a new terminal and change directory into client folder

```
	cd client && npm install
```
and run

```	
	npm run start
```
